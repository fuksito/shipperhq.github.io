Convert the cart into an order.
(Requires use of the x-shipperhq-secret-token header.)

```graphql
placeOrder(
  orderNumber: String!
  totalCharges: Float!
  carrierCode: String!
  methodCode: String!
  recipient: AddressInput
): Int!

```

### Arguments

#### `orderNumber` ([`String`](../rate-manager-service/scalars/string))

#### `totalCharges` ([`Float`](../rate-manager-service/scalars/float))

#### `carrierCode` ([`String`](../rate-manager-service/scalars/string))

#### `methodCode` ([`String`](../rate-manager-service/scalars/string))

#### `recipient` ([`AddressInput`](../rate-manager-service/inputs/address-input))

### Type

#### [`Int`](../rate-manager-service/scalars/int)

The `Int` scalar type represents non-fractional signed whole numeric values. Int can represent values between -(2^31) and 2^31 - 1.
