Fetch an Authentication secret.
(Use with the x-shipperhq-secret-token header to authenticate requests.)

```graphql
createSecretToken(
  api_key: String!
  auth_code: String!
): SecretToken!

```

### Arguments

#### `api_key` ([`String`](../../rate-manager-service/scalars/string))

#### `auth_code` ([`String`](../../rate-manager-service/scalars/string))

### Type

#### [`SecretToken`](../../rate-manager-service/objects/secret-token)

An authentication token returned from createSecretToken
