title: resetShippingChosen
---

Reset all cart options to their original values.
(Requires use of the x-shipperhq-secret-token or x-shipperhq-public-token headers.)

```graphql
resetShippingChosen: Int!

```

### Type

#### [`Int`](../../rate-manager-service/scalars/int)

The `Int` scalar type represents non-fractional signed whole numeric values. Int can represent values between -(2^31) and 2^31 - 1.
